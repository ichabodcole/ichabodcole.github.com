A Few Things I'm Working On

BinauralJS

A Javascript/CoffeeScript library for generating binaural beats using the Web Audio API More on binaural beats Here

context = new webkitAudioContext()
     bn1 = new BinauralBeat(context, frequency, beatRate)
     bn1.setVolume(0.5)
     bn1Node = bn1.getNode()
     bn1Node.connect(context.destination)

NoiseGenJS

A Javascript/CoffeeScript library for generating several types of ambient noise using the Web Audio API.[DEMO](http://htmlpreview.github.com/?https://github.com/ichabodcole/NoiseGenJS/blob/master/example/app.html)

context = new webkitAudioContext()
     ng1 = new NoiseGen(context, [type="brown", "white", "pink"])
     ng1.setVolume(0.5)
     ng1Node = ng1.getNode()
     ng1Node.connect(context.destination)
